Definitions

Nomenclature of CFCs, HCFCs:

     (H)CFC-x x x
      |     | | |_____ Number of Fluorine atoms
      |     | |_______ Number of Hydrogen atoms + 1
      |     |_________ Number of Carbon atoms   - 1
      |
      |                Chlorine atoms are not counted
      |
      |____ indicates alternative Fluorocarbons (HCFC)
     
    Example: CFC-12 (also known as F-12)  is CF2Cl2
 
 

Nomenclature of Halons:

     Halon-x x x x
           | | | |___ Number of Bromine atoms
           | | |_____ Number of Chlorine atoms
           | |_______ Number of Fluorine atoms
           |_________ Number of Carbon atoms
    Example: Halon-2402  is C2F4Br2
